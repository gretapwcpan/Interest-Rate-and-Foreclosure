# Project Description
This small program is to directly fetch the data from the Taiwanese government, including Taiwanese Central Bank and Ministry of the Interior,
to present the interplay between interest rate, loan burden ratio and foreclosure movement for over 15 years.

# Main library
- selenium (Web Crawler)
- re pandas numpy(Regular Expresssion for data processing/reports integration)
- matplotlib (Visualization)

# Raw Data Source
- Central Bank https://www.cbc.gov.tw/tw/np-369-1.html
- Ministry of the Interior https://pip.moi.gov.tw/V3/A/SCRA0601.aspx
- See sample of raw data in bankrate.xls
